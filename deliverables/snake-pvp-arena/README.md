# Snake PvP Arena

一个全栈 Web 应用，包含用户注册/登录、认证后访问贪吃蛇游戏、实时在线双人 PvP 对战、自动匹配、持久化积分和对战结果追踪。

## 技术方案

- **前端**: React 18 + TypeScript + Vite + TailwindCSS + Canvas 2D 游戏渲染
- **后端**: Express + Socket.IO + TypeScript + JWT 认证
- **持久化**: 本地 JSON 文件（FileDatabase）
- **对战模型**: 服务端权威 tick（150ms），客户端只发送方向输入
- **匹配系统**: 内存队列自动匹配 + 手动房间码

## 功能概览

- 用户注册/登录（JWT token 认证，表单验证）
- 路由守卫（未登录自动跳转登录页）
- 游戏大厅（在线人数、快速匹配、房间码创建/加入）
- 实时 PvP 对战（Canvas 渲染、方向键/WASD/触屏控制）
- 对战结束显示胜负结果和积分变化
- 个人战绩页（最高分、胜负统计、胜率、最近对局）
- 排行榜（按胜场排序，支持分页）
- 吃食物粒子特效

## 快速启动

```bash
cd deliverables/snake-pvp-arena
npm run install:all
```

终端 1 — 启动后端:

```bash
npm run dev:server
```

终端 2 — 启动前端:

```bash
npm run dev:client
```

默认地址:

- 前端: `http://localhost:5173`
- 后端: `http://localhost:5001`

## 人工烟测路径

1. 打开 `http://localhost:5173`，未登录应自动跳转到登录页。
2. 点击"注册"，创建账号 A（如 user1/123456）。
3. 新开浏览器标签或无痕窗口，注册账号 B（如 user2/123456）。
4. **快速匹配**: 两个账号分别点击"快速匹配"，服务端自动配对，双方进入对战。
5. **房间码对战**: 账号 A 点击"创建房间"，复制房间码；账号 B 输入房间码加入。
6. 使用方向键/WASD 控制蛇的方向。吃到食物时有粒子特效。
7. 任一方碰壁/碰蛇后，双方看到胜负结果和分数。
8. 点击"返回大厅"，然后查看"战绩"页面，确认积分和历史记录已更新。
9. 查看"排行榜"标签页，确认排行榜数据正确。
