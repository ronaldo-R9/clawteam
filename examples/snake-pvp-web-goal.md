# Snake PvP Web Goal

Use this goal when you want to recreate the same ClawTeam breakthrough setup
used in this session.

## Goal

Design and implement a website with registration and login, authenticated
access to a snake game, real-time online two-player PvP matches, and persistent
score plus match-result tracking.

## Deliverable

A polished playable arcade web experience with a clear lobby, authenticated
entry, real-time two-player snake gameplay, and persistent match history.

## Success Criteria

- registration and login work end to end
- unauthenticated users cannot access the game area
- two online players can join the same match and play simultaneously
- scores and match results are stored persistently
- core flows are usable on desktop browsers

## Constraints

- start from a clean repository
- prefer a simple architecture that can be shipped quickly
- keep the first implementation pragmatic before polishing
- unsupported upside claims must be stated as hypotheses, experiments, or future work

## Breakthrough Target

Ship a polished playable arcade web experience, not just auth plus a local
snake clone. Make the lobby and match history clear enough to feel like a small
real product.
